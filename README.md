## :computer: Projeto

GoRestaurant é um aplicativo React Native usando TypeScript que permite ao cliente fazer pedidos, salvar como favorito e visualizar pedidos anteriores.

  <img src="https://res.cloudinary.com/w7apps/image/upload/v1605800452/gorestaurant-mobile_lg6tg6.gif" width="250">

## :rocket: Construído com

Este projeto foi desenvolvido com as seguintes tecnologias:

-   [React Native](https://facebook.github.io/react-native/)
-   [TypeScript](https://github.com/microsoft/TypeScript)
-   [Context API](https://reactjs.org/docs/context.html)
-   [Styled-components](https://www.styled-components.com/)
-   [Json Server](https://github.com/typicode/json-server)
-   [React Navigation](https://reactnavigation.org/)
-   [React Native Vector Icons](https://github.com/oblador/react-native-vector-icons)
-   [Axios](https://github.com/axios/axios)
